# Cointegration Pairs Trading Strategy

This project investigates whether AAPL, MSFT, and GOOG share a long-run equilibrium relationship using Johansen cointegration tests and a Vector Error Correction Model.

A mean-reversion trading strategy is implemented based on deviations from the equilibrium spread.

## Methods
ADF Unit Root Test  
KPSS Test  
Johansen Cointegration Test  
Vector Error Correction Model  

## Results
The strategy produced a positive Sharpe ratio after transaction costs.

## Tools
RStudio
Econometric time series methods (ADF, KPSS, Johansen cointegration, VECM)
